# communitywebsite

## Build Setup

```sh
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

## Previewing Locally

After generating the static project

```sh
$ npm install -g serve
$ serve dist
```
